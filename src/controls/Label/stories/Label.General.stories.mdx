## Image

A `Label` can contain an `Image`. Images are sized to match the Label height.

export const ImageTemplate = (args) => 
<div>
  <Label {...args}/>
  <Label size="small"><Image src='https://react.semantic-ui.com/images/avatar/small/elliot.jpg'/>Two</Label>
  <Label size="huge"><Image src='https://react.semantic-ui.com/images/avatar/small/stevie.jpg'/>Three</Label>
</div>
