## clearable

A `clearable` `Input` presents a clickable icon that clears the input's contents. 
The icon only appears when the `Input` has content. The Input must be a 
controlled component for this to work.

## date

An `Input` control of type `date` allows date picking. 

* Use `nofuture` to disallow only future dates. 
* The `format` attribute controls how time is formatted in the input display (defaults to `HH:mm:ss`).

See https://date-fns.org/v2.21.1/docs/format for formatting.

<Story
  name="date"
  args={{ type:'date', format: "dd-MM-yyyy", placeholder: "Select date"  }}>
  {Template.bind({})}
</Story>

## time

An `Input` control of type `time` can show time in 12 or 24 hours, and optionally add a clock face.

* Hours and minutes are always shown, but seconds are optional and appear only if `seconds` is set. 
* If the optional property `clock` is given, then a clock face is shown.
* The `format` attribute controls how time is formatted in the input display (defaults to `HH:mm:ss`).

See https://date-fns.org/v2.21.1/docs/format for formatting.

<Story
  name="time"
  args={{ type:'time', placeholder: "Select time", clock: true }}>
  {Template.bind({})}
</Story>
