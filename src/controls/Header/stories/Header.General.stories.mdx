## Size

Headers can have `size="h1"` through `size="h6"`. The `size` attribute is required.

By default, sizes are absolute. If `relative` is specified, then sizes are relative to the container font size.

export const SizeTemplate = (args) => 
<div>
  <Header {...args}>My header</Header>
  <Header size="h1">Size h1</Header>
  <Header size="h2">Size h2</Header>
  <Header size="h3">Size h3</Header>
  <Header size="h4">Size h4</Header>
  <Header size="h5">Size h5</Header>
  <Header size="h6">Size h6</Header>
</div>

<Canvas>
  <Story
    name="Size"
    args={{ size: 'h3' }}>
    {SizeTemplate.bind({})}
  </Story>
</Canvas>

## Icon

An `icon` header emphasizes its icon.

  <Header size="h1" icon><Icon name="at" circular/> Icon header</Header>
</div>

# Image
