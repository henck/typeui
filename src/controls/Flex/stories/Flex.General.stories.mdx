## Column width

The `width` attribute of `Flex.Column` can be used to give weight to a column.

export const ColumnTemplate = (args) =>

## Flex.Divider 

A `Flex.Divider` can be added between two columns. This *only* works for exactly two columns. 
The divider automatically becomes horizontal when the screen is narrow enough and the Flex is `stackable`. 
An optional text can be placed in the divider element. 

export const DividerTemplate = (args) =>
<Flex {...args}>
  <Flex.Divider>and</Flex.Divider>
  <Flex.Row>
    <Flex.Column width={1}>
      <Box color="khaki">One</Box>
    </Flex.Column>
    <Flex.Column width={1}>
      <Box color="Lavender">Two</Box>
    </Flex.Column>
  </Flex.Row>
</Flex>

## Flex.Quick 

A `Flex.Quick` is used to vertically-align content in a row, and putting small spacing between items. 
Optionally, an `align` prop of `center`, `top` or `bottom` can be provided. The default is `center`.

export const QuickTemplate = (args) => 
<Flex.Quick {...args}>
  <Icon name="code"/>Hello
</Flex.Quick>

## Flex.Row 

There can be multiple `Flex.Row` elements, but there is no vertical connection 
between cells (this is not a grid).

Note that to obtain a grid-like flex, it's necessary to set `width` on columns.
Without it, columns will not flex.

export const RowTemplate = (args) =>
<Flex {...args}>
  <Flex.Row>
    <Flex.Column>
      <Box color="goldenrod">One</Box>
    </Flex.Column>
    <Flex.Column>
      <Box>Two</Box>
    </Flex.Column>
  </Flex.Row>
  <Flex.Row>
    <Flex.Column width={1}>
      <Box color="skyblue">Three</Box>
    </Flex.Column>
    <Flex.Column width={1}>
      <Box color="LightSalmon">Four</Box>
    </Flex.Column>
  </Flex.Row>
</Flex>

## Flex.Columns 

Use the `Flex.Columns` shortcut to quickly create multiple columns. Use `count` 
to specify the desired number of columns. The columns will stack automatically 
on a small screen.

export const ColumnsTemplate = (args) =>
<Flex.Columns count={2}>
  <Box color="goldenrod">One</Box>
  <Box>Two</Box>
  <Box color="skyblue">Three</Box>
  <Box color="LightSalmon">Four</Box>
</Flex.Columns>
