## width 

A `Dialog` has a default width in pixels. On a large screen the width defaults
to _600_ pixels, while on small screen, it defaults to _400_ pixels (done through 
a `@media` query). The `width` attribute can be used to override this.

export const Template = (args) => 
<div style={{height: '300px'}}>
<Dialog {...args}>
  <Dialog.Header>Welcome</Dialog.Header>
  <Dialog.Content>Good day!</Dialog.Content>
  <Dialog.Footer>
    <Button>OK</Button>
  </Dialog.Footer>
</Dialog>
</div>

## Scrolling content

The `Dialog.Content` element will add a vertical scrollbar if the height of 
the content exceeds _70%_ of the vertical screen size. The `maxHeight` property
can be used to customize this by specifying a different percentage.

## Footer alignment

The `Dialog.Footer` element accepts an optjonal `align` property that behaves 
like Flexbox's `justify-content`, facilitating the placement of buttons in the 
footer. Alignment defaults to the right for all items (`justify-content: end`).

export const FooterTemplate = (args) => 
<div style={{height: '300px'}}>
<Dialog {...args}>
  <Dialog.Header>Welcome</Dialog.Header>
  <Dialog.Content>Good day!</Dialog.Content>
  <Dialog.Footer align="space-between">
    <Button primary>OK</Button>
    <Button secondary>Another button</Button>
  </Dialog.Footer>
</Dialog>
</div>
