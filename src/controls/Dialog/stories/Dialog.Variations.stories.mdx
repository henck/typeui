## width 

A `Dialog` has a default width in pixels, but the `width` attribute can be 
used to override that.

export const Template = (args) => 
<div style={{height: '300px'}}>
<Dialog {...args}>
  <Dialog.Header>Welcome</Dialog.Header>
  <Dialog.Content>Good day!</Dialog.Content>
  <Dialog.Footer>
    <Button>OK</Button>
  </Dialog.Footer>
</Dialog>
</div>
