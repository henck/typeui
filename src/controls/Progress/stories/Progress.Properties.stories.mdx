# Progress

The `Progress` component shows a progress bar, filled to a percentage equal to 
`value`. The bar always stretches to fill all horizontal space available to it.

<ArgsTable of={Progress}/>

## Example

A default `Progress` shows a transparent bar, filled to `value` percent. It
will have the primary theme color, unless overridden using the `color` 
property.
