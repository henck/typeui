# CircularProgress

A `CircularProgress` shows a percentage value, with a circular line around it.
Optionally it may be `animated` so that the line grows from 0% to a target 
value. There are various properties that control the `CircularProgress`'s 
appearance.

## Default CircularProgress

All properties are optional, except `value`.
