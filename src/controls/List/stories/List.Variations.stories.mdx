## animated

A `List` can be `animated` to set the hovered item apart from the list.

<Canvas>
  <Story
    name="animated"
    args={{ align: 'center', animated: true }}>

  </Story>
</Canvas>

## bulleted 

A `bulleted` list marks items with a bullet.

You can provide a bullet style type using `type` (default `disc`).

<Canvas>
  <Story
    name="bulleted"
    args={{ bulleted: true, type: 'circle' }}>
    {Template.bind({})}
  </Story>
</Canvas>

## horizontal 

A `horizontal` `List` lines up its items horizontally.

A `relaxed` `List` adds more negative space around its items. Optionally `relaxed="very"`.

A `divided` `List` adds a horizontal line between list items.

  <Story

## selection 

A `selection` list shows a selection rectangle on hover. This works well with `onClick`. 
   
If `divided` then the selection rectangle has no corner radius.

  <Story
    name="selection"
    args={{ selection: true, divided: false }}>

  </Story>
</Canvas>
