# List

A `List` shows a list of `List.Item` elements, by default vertically (but a 
horizontal List is possible). List items can have an `Icon` and `Image`. 

<ArgsTable of={List}/>

## Standard List

A standard `List` has no bullets, icons or images. Its items are `List.Item` elements.
