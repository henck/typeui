# Button: Groups

The `Button.Group` component serves to group `Button` instances together. They
can be grouped horizontally or vertically, and a `Button.Or` can further be used
to separate them.

<ArgsTable of={Button}/>

## Attributes

Groups can share attributes like `color`, `size`, `compact`, `basic` and `icon`.
