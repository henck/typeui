## icon

A `Button` can have an `<Icon>` inside it. The icon will be 80% opaque, and 
100% opaque when the button is hovered. If the `icon` attribute is specified, 
then the button will have no padding. The icon itself can have its own 
attributes.

The icon color will automatically adjust to the button's color.

export const IconTemplate = (args) => <div>
  <div>
    <Button {...args}><Icon name="circle" size="large"/> Button</Button>
  </div>
  <Divider hidden/>
  <div>
    <Button><Icon name="circle" size="large"/> Button</Button>
    <Button><Icon name="circle" size="large"/></Button>
    <Button icon><Icon name="circle" size="large"/></Button>
  </div>
  <Divider hidden/>
  <div>
    <Button basic><Icon name="circle" size="large"/> Button</Button>

  <Divider hidden/>
  <div>

    <Button basic color="skyblue"><Icon name="circle" size="large"/></Button>

  </div>            
</div>

## disabled

A `disabled` button cannot be interacted with. `onClick` event handlers 
registered on it will not fire.
