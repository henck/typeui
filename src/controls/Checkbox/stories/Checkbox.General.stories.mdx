# Checkbox: General 
.

## Label

A `Checkbox` should (usually) have a `label`. This is not the same as a 
`Form.Field` label. The label is clickable and toggles the checkbox.

<Canvas>
  <Story
    name="label"
    args={{label: "My label"}}>
    {Template.bind({})}
  </Story>
</Canvas>

## radio 

A radio group must provide the `radio` attribute to each of its members,
and all must have the same `name`, and provide a `value`.

For `radio` buttons, `type` will default to `circle` unless overridden.

export const RadioTemplate = (args) => <div style={{display: 'flex', flexDirection: 'column'}}>
  <Checkbox {...args}/>
  <Divider hidden/>
  <Checkbox name="myradio" radio value={2} label="Two"/>
  <Divider hidden/>
  <Checkbox name="myradio" radio value={3} label="Three"/>
</div>

<Canvas>
  <Story
    name="radio"
    args={{radio: true, label: "One", value: 1, name: "myradio" }}>
    {RadioTemplate.bind({})}
  </Story>
</Canvas>
